Módulo 1¶

modulo1.beta_equation(pf)¶

This function calculates the reliability index value for a given probability of failure (pf).

Parâmetros:

pf (float) – Probability of failure (pf) value to be converted to beta.

Levanta:

ValueError – If pf is not between 0 and 1.

Retorno:

Beta value (β) corresponding to the given pf.

Tipo de retorno:

float | str

modulo1.calc_pf_beta(df_or_path, numerical_model, n_constraints)¶

Calculates the probability of failure (pf) and reliability index (β) based on the columns of a DataFrame that start with “I” (indicator function). If a .txt file path is passed, this function evaluates pf and β values too.

Parâmetros:

  • df_or_path (DataFrame | str) – A DataFrame containing boolean indicator columns prefixed with “I”, or a string path to a .txt file.

  • numerical_model (str) – Dictionary containing the numerical model.

  • n_constraints (int) – Number of limit state functions or constraints.

Retorno:

Tuple of DataFrames:

  • df_pf: probability of failure values for each column prefixed with “G”.

  • df_beta: reliability index values for each column prefixed with “G”.

Tipo de retorno:

tuple[DataFrame, DataFrame]

modulo1.cornell_algorithm_structural_analysis(setup)¶

Applies the Cornell reliability algorithm to evaluate the probability of failure and reliability index.

Parâmetros:

setup (dict) –

Dictionary containing the input settings, including:

  • ”variables settings”: list of variable configurations with “mean” and “sigma” values,

  • ”objective function”: function that returns a list of g values,

  • ”none variable”: additional fixed parameters for the objective function.

Retorno:

Tuple of two DataFrames:

  • pf_dataframe: DataFrame containing the probability of failure (pf) for each limit state.

  • beta_dataframe: DataFrame containing the reliability index (β) for each limit state.

Tipo de retorno:

tuple[DataFrame, DataFrame]

modulo1.newton_raphson(f, df, x0, tol)¶

Calculates the root of a function using the Newton-Raphson method.

Parâmetros:

  • f (Callable) – Function for which the root is sought.

  • df (Callable) – Derivative of the function.

  • x0 (float) – Initial guess for the root.

  • tol (float) – Tolerance for convergence.

Retorno:

Approximated root of the function.

Tipo de retorno:

float

modulo1.pf_equation(beta)¶

Calculates the probability of failure (pf) for a given reliability index (β), using the cumulative distribution function (CDF) of the standard normal distribution.

Parâmetros:

beta (float) – Reliability index (β).

Retorno:

Probability of failure.

Tipo de retorno:

float

modulo1.sampling(n_samples, model, variables_setup)¶

Generates a set of random numbers according to a specified probability distribution model.

Parâmetros:

  • n_samples (int) – Number of samples to generate.

  • model (dict) – Dictionary containing the model parameters.

  • variables_setup (list) – List of dictionaries, each containing parameters for a random variable.

Retorno:

Numpy array with the generated random samples.

Tipo de retorno:

ndarray
