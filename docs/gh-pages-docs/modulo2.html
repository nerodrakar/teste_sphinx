Módulo 2¶

modulo2.analisar_dados(dados)¶

Analisa uma lista com estrutura fixa: [float, str, dict].

Parâmetros:

dados (list[float | str | dict[str, int | bool]]) – Lista com exatamente 3 elementos: - posição 0 (float): valor numérico. - posição 1 (str): descrição textual. - posição 2 (dict): dicionário com as chaves “id” (int) e “valido” (bool).

Levanta:

ValueError – Se a estrutura da lista estiver incorreta.

Retorno:

Texto formatado com o conteúdo processado.

Tipo de retorno:

str

Exemplo de uso:

>>> dados = [7.5, "Tensão limite", {"id": 101, "valido": True}]
>>> resultado = analisar_dados(dados)
>>> print(resultado)
Tensão limite: 7.50 (ID=101, válido=True)
modulo2.calcular_estatisticas(valores, wander)¶

Calcula estatísticas básicas a partir de uma lista de valores numéricos.

Parâmetros:

  • valores (list[float]) – Lista de números reais.

  • wander (str)

Levanta:

ValueError – Se a lista estiver vazia.

Retorno:

Uma tupla contendo:

  • média (float)

  • valor mínimo (float)

  • valor máximo (float)

Tipo de retorno:

tuple[float, float, float]

modulo2.processar_dados(dados)¶

Processa um dicionário contendo informações de um usuário.

Parâmetros:

dados (dict[str, str | int | bool]) –

Dicionário com as seguintes chaves obrigatórias:

  • nome (str): Nome completo do usuário.

  • idade (int): Idade do usuário em anos.

  • ativo (bool): Indica se o usuário está ativo.

Levanta:

KeyError – Se alguma das chaves obrigatórias estiver ausente.

Retorno:

Uma string formatada com os dados processados.

Tipo de retorno:

str
